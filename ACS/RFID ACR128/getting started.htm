System Requirements

This kit will require the following system specifications:

- Personal Computer with 800 MHz Processor or higher;
- Microsoft Windows: 2000 / 2003 /  XP / VISTA;
- CD-ROM Drive;
- 500 MB Free Storage space;
- One available USB port.

Note:

If you have installed previous version of ACR128 Reader SDK, then you must uninstall these version via Add/Remove Programs in Control Panel before installing this SDK version. Make sure that all folders have been deleted.

For non VISTA users, you need to install the .NET Framework to be able to use the ACR128U Tool and Demo Applications. This can be downloaded via: 

Microsoft .NET Framework Version 2.0 (x86) 

http://www.microsoft.com/downloads/details.aspx?FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5&displaylang=en

 

Microsoft .NET Framework Version 2.0 (x64)

SDK Checklist

The ACR128 Dual Interface Reader SDK contains the following:

- 1 unit ACR128 Dual Interface Reader;
- 1 SDK CD-ROM;
- 1 ACOS6-SAM Card
- 5 MiFare 1K Cards and 5 ACOS3-24K Cards
 

A. Connection Diagram

To connect the ACR128 to your PC, plug in the USB connector (Type A) into available USB Port.

 Install SDK

Insert the ACR128 SDK CD-ROM into your CD-ROM drive. If the screen below does not appear, run x:\ACR128SDK.EXE where x is the drive letter of your CDROM.

 

Install SDK Components

The default path where the SDK will be installed is x:\Program Files\Advanced Card Systems Ltd\ACR128 Dual Interface Reader SDK where x is the drive letter of your local Windows drive. Click here to view the SDK Installation Guide.

Install SDK Components button from the ACR128 SDK Setup GUI. It includes the following:

o Multi-Application Demo – This demonstration program emulates several real world scenarios where ACR128 can be used.

o Sample Codes - Sample codes written in different programming language are provided, namely Java, Delphi 7, Visual Basic 6, Visual C++ 6, Visual C++ 2005 (x64), Visual C # 2005 and Visual Basic .Net 2005.

The sample codes show the different capabilities of the ACR128 and showcases how to control the ACR128 Peripherals and how to communicate with a contact and contactless cards.

o Tools and Utilities

o ACR128 Card Tool – The ACR128 Card Tool is an application utility tool that enables the user to perform reader and contactless card related commands.

o ACS Easy Key - The ACS Easy Key is a utility program that will allows you to change the MiFare Security Settings.

o ACS Card Tool – The ACS Card Tool is designed to work with ACR128 in using the ACOS cards.

o QuickView - The ACS Quickview is a utility program that will check if you have properly installed the ACR128 device.

o User Manuals and Reference Materials - These files are in PDF. You can install the Acrobat Reader from the ACR128 SDK Setup GUI or just click this and run it.

o ACR128 SDK User Manual – This manual educates the user on how to install the SDK and gives a brief description of the components included in the SDK. Read it.

o ACR128 API – contains the commands that can be used to create applications using the ACR128 Dual Interface Reader. Read it.

o ACR128 Technical Specifications – physical and technical specifications of ACR128. Read it.

o Mifare 1K Card Reference Manual – specifications for the 1K Mifare Contactless Card. Read it.

o Mifare 4K Card Reference Manual – specifications for the 4K Mifare Contactless Card. Read it.

o ACOS3 Reference Manual – specifications for the ACOS3 MCU Card. Read it.

o  ACOS6 SAM Reference Manual – specifications for the ACOS3 MCU Card. Read it.
