System Requirements

This kit will require the following system specifications:

  • IBM compatible Personal Computer with Intel Pentium 800 MHz Processor or higher;
  • Microsoft Windows 2000 / XP / 2003 / Vista;
  • CD-ROM Drive;
  • One available USB port.

SDK Checklist

This kit contains the following:

  • 1 unit ACR38 CCID Smart Card Reader;
  • 1 unit ACR38T-IBS CCID SIMTracker;
  • 1 unit ABR08 Balance Reader;
  • 5 pieces ACOS3 T=0 Smart Cards;
  • 5 pieces ACOS3 SIM-sized Smart Cards;
  • 5 pieces SLE5528 Memory Cards;
  • 5 pieces SLE5542 Memory Cards;
  • ACR38 CCID SDK CDROM.

Connection Diagram

Note: You can connect the ACR38 CCID Smart Card Reader/Writer device to your PC anytime AFTER the drivers has been loaded.

Install SDK

Insert the ACR38 CCID SDK CD into your CDROM drive. If the screen below does not appear, run x:\autorun.exe where x is the drive letter of your CDROM.  

SDK Components

The default path where the SDK will be installed is x:\Program Files\Advanced Card Systems Ltd\ACR38 CCID Smart Card Reader SDK where x is the drive letter of your local Windows drive. Click here to view the SDK Components Installation Guide.

The SDK Components includes the following:

Sample Applications – Demo programs that uses the ACR38 CCID for various applications.

Casino Demo – uses ACOS card and ACR38 CCID reader in a Casino environment. The card is used to store the player’s points when playing games.

School Demo – uses ACOS card and ACR38 CCID reader in a School environment. The card serves as school ID, library card, cash card, and many more.

Sample Codes - This is a collection of sample ACR38 CCID PC/SC programs with source codes for the user to quickly learn how to develop their own applications. It is written in various programming languages emphasizing certain features of the ACR38 CCID Smart Card Reader and/or the smart cards included in the kit.

The sample codes are written in the following programming languages: Microsoft Visual Basic, Visual C++, Visual C++ 2005 (x64), VB.NET, C# , Borland Delphi and Java. It includes:

ACOS3 – demonstrates how to work with ACOS3 cards, such as power-up, create/read/write files and mutual authentication commands;

MEMORY CARDS – demonstrates how to access the following memory cards:

  • I2C
  • SLE4418
  • SLE4428/5528
  • SLE4432/5542

    • *Note that SLE4428 sample codes work with SLE5528 cards

OTHERS – demonstrates other command types, such as how to check if a card is inserted and how to get the card’s Answer-To-Reset string.

Tools and Utilities

CardTool -  is a utility program that allows you to send commands to any PC/SC-compliant smart card reader, and to any ISO7816 T=0 compliant smart card. You can send commands to your ACR38 smart card reader using this tool. CardTool supports the following ACS readers: ACR30U, ACR30S, ACR30SP, ACK30S, ACT30, ACR38, ACR38 CCID, ACR38T, ACR38DT, ACR38ET, ACR38K, ADT60, AET60, AET63, ACR80 and ACR88.

PCSC Learning Tool -    is a utility program that allows you to send commands to any PC/SC-compliant smart card reader, and to any ISO7816 T=0 compliant smart card. It teaches you how to use the PC/SC APIs step-by-step and what parameters to use.

QuickView - is a utility program that checks if you have properly installed your ACS readers.

Scripting Tool - is a utility program that allows you to communicate with a smart card by means of a script command file.

ACR38 CCID Reference Manual -Specifications on how to directly program the ACR38 CCID device. . Read it.

ACR38 CCID PC/SC Memory Card Access Manual - Reference Manual for Memory Card Support using PC/SC driver. Read it.

ACR38 CCID Technical Specifications  -Physical and Technical specifications for the ACR38 CCID smart card reader. Read it.

ACR38T-IBS CCID Technical Specifications - Physical and Technical specifications for the ACR38T SIMTracker.  Read it.

Balance Reader Technical Specifications - Physical and Technical specifications for the ABR Balance Reader Series. Read it.

ACOS3 Reference Manual  - Reference / Programming manual for the ACOS3 T=0 Smart Card. It lists all commands that can be sent by the application to the ACOS3 card. Read it.

Training Materials - Introductory reference to smart card technology. It discusses the components of the smart card, its categorization, history, standards and its applications. Also, it gives information on different types of memory cards, smart card chip operating system platform and the PC/SC workgroup. Open it.
